ADEGuard: Adverse Drug Event (ADE) Detection and Severity Analysis

A Hybrid Transformer-Based NLP System

ADEGuard is an intelligent clinical text analytics system that automatically detects Adverse Drug Events (ADEs) and evaluates their severity from unstructured narratives. It combines BioBERT-based NER for ADE and drug extraction, a hybrid severity classifier (rule-based + BioBERT fine-tuning with Snorkel weak labels), and Sentence-BERT embeddings for clustering and similarity analysis. Using t-SNE + K-Means, it visualizes ADE patterns across age and severity groups, while SHAP explainability ensures transparency in model predictions. The interactive Streamlit dashboard features token-level entity highlights, severity adjudication, cluster visualizations, and downloadable clinical insights for pharmacovigilance and decision support.
